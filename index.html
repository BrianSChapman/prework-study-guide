HTML

  • The head element contains information about the webpage
  • The meta element informs the browser how to render the information given
  • The body element represent the visible content shown to the user
  • h1-h6 elements are headers used to draw attention to text on the page
  • The p element is a paragraph, automatically adding single blank lines before & after the added element
  • The ul, ol, and li elements are unordered list, ordered list, and list item (Like I'm using now)
  • The img element is to toss a photo on your page. Note that the src attribute defines the location of the image
  • The br element is to create a line break between two blocks of content
  • Here's a cheat sheet for more basic HTML tips & tricks.

CSS

  • There are (3) ways to style a webpage using CSS: Inline CSS, internal CSS style sheet,and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes act as a CSS rule that can be used to designate certain elements (with associated class selector) that are affected by CSS styling while others are not
  • We can style HTML elements using the elements themselves as selectors or using class attributes as selectors (To affect multiple elements at once).
  • Here's a detailed Cheat sheet on more ways to add CSS flair.

Git

  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git clone [url] : Retrieves an entire repo from a hosted location via URL
  • git branch: Lists your branches with a * appearing next to current branch
  • git status: Shows changes you've made but haven't commited yet
  • git commit -m "[Description]": Commit content you have staged & leaving yourself a comment on what changes were made
  • Here's a cheat sheet with the most common Git commands.
  • Memorize and internalize the GitHub flow to become more efficient.

JavaScript

  • A variable (var) is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes a code in a script (Order Matters.)
  • Control flow can be interrupted by using conditional statements and loops to control which code, if any, will be executed.
  • Falsy = 00 -0, On, "", null, undefined, NaN, or value of False.
  • Truthy is all other values, including a Boolean (T/F) with the value of true.
  • An Array is a single variable that is used to hold a group of data (Typically related in some way). Since the contents of the array become indexed, it makes testing different values much easier.
  • Javascript is an intimidating topic, so use the modules and this Basic Guide to help acclimate you.
